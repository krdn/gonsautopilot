# GonsAutoPilot

Claude Code 플러그인 기반 풀스택 자동 테스트/배포 시스템

## 프로젝트 구조

- `plugin/` — 플러그인 코어
  - `skills/` — 사용자 호출 스킬 (`/gonsautopilot`, `/gonsautopilot:test` 등)
  - `agents/` — 전문 Agent 정의 (orchestrator, test, build, deploy, monitor)
  - `hooks/` — git hook 스크립트
  - `lib/` — 공유 셸 유틸리티
  - `state/` — 런타임 상태 저장소 (pipeline.json 등)
- `configs/` — 설정 템플릿
- `docs/plans/` — 설계 문서

## 핵심 명령어

```bash
/gonsautopilot              # 전체 파이프라인
/gonsautopilot:test         # 테스트만
/gonsautopilot:deploy       # 빌드+배포만
/gonsautopilot:status       # 상태 조회
/gonsautopilot:rollback     # 수동 롤백
```

## 개발 규칙

- 셸 스크립트는 `set -euo pipefail` 사용
- JSON 처리에 `jq` 사용
- YAML 처리에 `yq` 사용
- 상태 파일 수정 시 반드시 `state-manager.sh`를 통해 수행
- Agent 이름 접두사: `gap-` (GonsAutoPilot)
